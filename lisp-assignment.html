History of Lisp:

Orginally invented in 1958, by John McCarthy at MIT. Lisp is the second-oldest high-level programming language in widespread use today. Only Fortran is older by 1 year. It was originally created as a mathematical notation for computer programs. It quickly became the favored programming language for artificial intelligence research. The name LISP derives from "LISt processor". Linked lists are one of Lisp's major data structures, and Lisp source code is made of lists.

Purpose:

Lisp pioneered many ideas in computer science, including tree data structures, automatic storage management, dynamic typing, conditionals, higher-order functions, recursion, the self-hosting compiler, and the read-eval-print loop.

Type:

Lisp is a compiled functional language that can be interpreted. The Lisp family of languages are all "strong typed" in the sense that typing errors are prevented at runtime.

Example:

(defparameter *small* 1)
(defparameter *big* 100)

(defun guess-my-number ()
(ash (+ *small* *big*) -1))

(defun smaller ()
(setf *big* (1- (guess-my-number)))
(guess-my-number))
(defun bigger ()
(setf *small* (1+ (guess-my-number)))
(guess-my-number))
(defun start-over ()
(defparameter *small* 1)
(defparameter *big* 100)
(guess-my-number))
